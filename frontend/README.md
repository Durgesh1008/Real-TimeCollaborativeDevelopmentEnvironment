# 🚀 CodeSync - Real-Time Collaborative Editor (Frontend)

CodeSync is a powerful, real-time collaborative code editor featuring live chat, voice calling, and AI-powered code fixing.

## ✨ Features
* **Real-time Collaboration:** Edit code with friends using Socket.io.
* **Multi-language Support:** JavaScript, Python, Java, and C++.
* **Live Audio Chat:** Integrated Voice-over-IP using PeerJS.
* **AI Code Assistant:** Get instant fixes and explanations for your code errors.
* **Terminal Output:** Run your code and see results instantly.

## 🛠️ Tech Stack
* **Framework:** React.js
* **Editor:** Monaco Editor (@monaco-editor/react)
* **Styling:** CSS3 (Custom Grid/Flexbox)
* **Real-time:** Socket.io-client
* **WebRTC:** PeerJS
